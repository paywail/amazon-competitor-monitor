# AI 分析指令模板

## 标准版指令（完整分析）

```
请你担任亚马逊竞品运营分析师，我会提供固定监控竞品的本周数据信息，请按照要求完成分析：

1. 逐条梳理每款竞品本周所有变动内容，包含售价、优惠券、促销活动、标题卖点、图片页面、BSR类目排名、评价口碑变化

2. 对每一处变动判定影响等级：无影响/轻微影响/中度冲击/高度威胁

3. 结合我方同款产品现状，给出具体、可落地、可直接执行的运营应对动作

4. 汇总本周整体竞争局势，标注风险点，列出下周重点监控方向

5. 最终整理成一页式竞品监控周报格式，条理清晰，结论明确，剔除冗余内容
```

## 精简速算版指令（快速出结果）

```
根据我给到的亚马逊竞品本周数据，整理变动明细，评估竞争影响，输出对应的调价、Listing优化、广告调整、排名卡位实操建议，汇总成简洁版竞品行动周报，建议区分紧急执行、常规跟进、持续观察三个优先级。
```

## 分析维度说明

### 价格分析

| 分析点 | 判断标准 |
|--------|----------|
| 价格区间 | 是否进入新价格区间（如7000日元以下） |
| 划线价 | 是否使用划线价制造折扣感 |
| 优惠券 | Coupon百分比、是否有促销叠加 |
| 价格趋势 | 与上周对比是否有变化 |

### 卖点分析

| 分析点 | 判断标准 |
|--------|----------|
| 核心卖点 | 尺寸、承重、收纳、组装、显示器臂等 |
| 卖点一致性 | 标题/五点/描述是否一致 |
| 新卖点 | 是否新增卖点描述 |
| 竞品卖点差距 | 我品是否覆盖竞品核心卖点 |

### 评价分析

| 分析点 | 判断标准 |
|--------|----------|
| 评分变化 | 评分是否下降（<4.2需警惕） |
| 评论增长 | 评论增长速度（老品vs新品） |
| 评论优势 | 山和直销级别评论碾压需长期积累 |

### 影响等级判定

| 等级 | 定义 | 判断标准 |
|------|------|----------|
| 🔴 高度威胁 | 直接影响转化和销量 | 价格战、大幅Coupon、评论碾压 |
| 🟠 中度冲击 | 可能分流部分用户 | 划线价、大尺寸差异化、新卖点 |
| 🟡 轻微影响 | 需关注但不紧急 | 企业促销、新品入场、配送方式 |
| ⚪ 无实质影响 | 对个人消费者影响小 | 高端企业产品、无Prime |

## 行动建议优先级

| 优先级 | 时间要求 | 适用场景 |
|--------|----------|----------|
| 🔴 立即执行 | 1-2天内 | 价格战跟进、Coupon监控、Listing紧急优化 |
| 🟡 本周跟进 | 本周内 | 卖点强化、划线价策略、广告调整 |
| 🟢 长期观察 | 持续 | 评论积累、产品线扩展、市场趋势 |

## 复盘指标

| 指标 | 定义 | 目标 |
|------|------|------|
| 动作执行率 | 建议动作执行数/总建议数 | ≥80% |
| 价格竞争力 | 我品价格 vs 竞品平均价格 | 处于前50% |
| 核心词排名 | 监控5-10个核心词排名变化 | 稳定或上升 |
| 评分维持 | 我品评分 | ≥4.2 |
